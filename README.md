\# House Price Prediction



\## Project Overview



This project predicts house prices using Machine Learning regression models.



The project compares:



\- Linear Regression

\- Random Forest Regressor



Dataset:

Housing Prices Dataset from Kaggle.



\## Features



\- Data Cleaning

\- One-Hot Encoding

\- Exploratory Data Analysis

\- Linear Regression

\- Random Forest

\- Model Evaluation

\- Feature Importance

\- Visualization



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Scikit-Learn

\- Matplotlib

\- Seaborn

\- Jupyter Notebook



\## Evaluation Metrics



\- MAE

\- RMSE

\- R² Score



\## Results



Random Forest performed better than Linear Regression in predicting house prices.
